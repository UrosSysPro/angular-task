# Angular Interview Task
## Compile and run
Requirements
- nodejs
- angular cli

Tested on:
- Windows 11
- Fedora Server 41
```
git clone https://github.com/UrosSysPro/angular-task.git
cd angular-task
npm install
npm start
```
